随着互联网行业的迅速发展，世界已经迈进了知识经济的时代，我国倡导建设人
才强国，因此人力资源成为了企业重要竞争力之一。一个企业想要取得成功，不仅需
要具备硬实力，对人力资源进行高效的管理也是必不可少的一部分。企业对人力资源
进行科学管理，对于企业、员工和人力资源部门都有益处。对于企业，可以随时查看
员工资料和查看统计信息等；对于人力资源部门，能够节省统计信息和业务操作时间；
对于企业员工，可以获得科学培训安排。因此研究企业的人力资源管理系统，具有现
实意义和计算机应用研究的意义。 
本文研究内容为某企业的人力资源管理系统的设计和开发。系统基于某企业人力
资源管理部门的现实业务需求，进行设计和开发。并且系统的适用对象为人力资源部
门的员工和企业高级管理者，特点是对人力资源操作员权限控制非常灵活且详细。系
统开发方法使用面向对象法，在文中对系统需求分析、系统设计、系统实现和系统测
试进行了论述。系统在开发中重要的一步是需求分析，本文使用用例图描述系统的角
色关系和业务需求，在确定开发需求合理且能实现的基础上，设计并实现了系统的员
工资料、人事管理、薪资管理和统计管理等功能。系统运用 Spring  Boot 和 Vue 框架
技术规范，使用 MySql 作为后台数据存储，开发出一个基于 B/S 模式的人力资源管理
系统。系统使用 IDEA、Code 和 Navicat 作为开发工具。为了保证系统的安全性，本
文在设计中使用 MD5 算法通过添加哈希计算中的随机散列值操作对操作员用户密码
进行加密。 
从论文的结构看，本文首先介绍了论文的研究背景，对国内外人力资源管系统设
计的优势和不足进行分析。由此确定研究内容具备研究意义后，为题目设计了研究目
标和方法。 
其次，本文对系统使用的相关关键技术进行介绍。 
再次，本文从需求分析、系统设计和系统实现三方面对系统设计和实现的过程进
行了论述。 
最后，本文对设计并实现的系统进行了展示和功能介绍，对系统进行了压力测试、
每个业务单元的功能测试和系统安全性测试。并且对本文中系统设计的不足和未来研
究展望。 
 
关键词：人力资源管理；Spring Boot；B/S；Vue

摘要 
现如今，传统的基于人工的人事管理已经难以解决一些复杂的文件管理、流
程管理等问题，久而久之便会给企业员工数据查找带来极大的困难。寿险公司人
力资源管理系统尤其在绩效考核，人力资源管理方面，对企业提升效益，加强管
理方面尤为重要，可以使得办公人员相对轻松的完成人力资源管理，从而使得企
业的人员管理工作更加系统化。寿险公司人力资源结构相对复杂，人员数量又相
对庞大，其人力资源管理系统需要完善的更新与开发。因此，本文以寿险公司为
例，对其人力资源管理系统进行研究，着重对人力资源管理系统中的绩效评价进
行全面分析。分析了目前寿险公司人力管理中在系统设置，组织机构、招聘管理、
培训管理、人事管理等流程中所存在的问题，并提出了改进的办法。利用 NET 支
撑平台的技术框架体系，采用 C#等编程语言，构建了基于 B/S 架构的各种特别的
模块，应用到寿险人力资源管理中。根据寿险公司针对人员工作管理的特殊要求
设计每一个模块。其中包括对绩效考核进行管理的模块、对人员薪资进行管理的
模块、及对员工各种培训进行管理的模块等。该系统该模块实现了寿险公司人力
资源管理的全流程和过程化监督。从而达到降低人寿保险公司人力资源管理成本、
提高管理效率的目的。

基于 Java 的企业人力资源信息系统的设计与实现
由于信息化技术的快速发展，企业的信息化技术建设也在发展中，而企业人
力资源管理是企业发展过程中相对重要的，通过合理的人力资源管理可以留住人
才。企业人力资源管理系统的开发就是为了使企业更好、更快速的发展，因此有
一个人力资源管理系统是不够的在以信息技术快速发展的情况下，人力资源在管
理工作中也在不断的推进。由于企业人力资源管理相对复杂，所以给人事管理人
员带来了很大的不便。人力资源管理首先就要建立人才信息库，然后再根据人才
库查询、管理等一系列的工作，因此使企业在管理上面得到了科学化、网络化、
标准化和信息化。
本系统是通过 Java 技术、SQL Server2008 数据库进行开发的，本系统主要
就是企业而开发的人力资源管理系统，开发本系统就是为了能够提高企业的管理
效率和人事管理的工作效率。
本文主要研究的就是企业人力资源管理系统，首先介绍了人力资源管理的选
题背景和研究的意义；再对系统进行需求分析，在需求分析阶段介绍了系统的业
务需求、功能用例分析、非功能性需求和可行性研究等内容；在系统设计过程中
主要介绍了系统的功能模块、业务流程、系统数据库设计等内容；在系统实现与
测试方面主要介绍了系统的界面实现，最后再进行了系统的测试。

摘    要 
随着企业运营成本的增加，当前中小型企业面临的压力越来越大，只有创造更
强的竞争力才能有更好的生存空间。降低企业运营的成本，提高执行的效率是加快
企业发展和提高竞争力的重要措施。在企业的运营成本中，人力资源成为了最活跃、
可增值因素，实现对人力资源的有效管理、提高人力资源的利用效率是企业成败的
关键。本文研究的目标是设计人力资源管理系统，着力提高人力资源管理的效率，
从而提高企业的竞争力，促进企业的发展。 
为更好地设计高效、实用的人力资源管理系统，在文中对人力资源管理相关知
识和理论基础进行了深入分析，为人力资源管理系统的设计提供更好知识支撑和理
论基础。为更好地选择人力资源管理系统的开发技术，对当前软件系统从软件建模、
数据库设计、软件编程设计、页面设计等技术展开分析，结合人力资源管理的需求，
选择合适的系统建模、功能设计与实现技术。在中小型企业中不同的行业对人力资
源管理有着不同的需求，为找出其对人力资源管理的共同需求，从不同角度对当前
中小型企业的人力资源情况进行分析，找出其共性的需求，结合本地中小企业的实
际情况进行系统建模，设计出符合实情的人力资源管理系统用例图。根据人力资源
系统用例图，确定系统的具体功能，设计出系统的整体功能图。 
在完成人力资源管理系统整体功能设计的基础上，为实现人力资源管理系统，
需要进一步对整体功能进行详细设计。首先对系统中的各个功能进行分析，使用时
序图对功能进行详细设计。再分析系统的数据需求，根据系统的数据需求使用 ER 模
型建立系统的数据模型，通过 ER 模型图设计出系统的数据表。 
为高效地实现人力资源管理系统，根据选择的 JSP 开发技术，搭建好 Tomcat 
Web 服务器，安装 SQL Server 数据库，建立人力资源管理系统数据库。根据系统功
能分析的结果，使用 JSP 编写代码实现系统功能，使用 CSS 与 DIV 语言设计系统
的页面。完成代码编写与页面设计后，部署系统的运行环境，进一步对系统展开测
试。在本文的最后对人力资源管理系统的开发过程进行回顾与总结，分析系统存在
问题，提出改进措施和展望软件系统发展的方向。
